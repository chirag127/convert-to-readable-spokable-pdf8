Frequently Asked Questions

How do I get a Gemini API key?

Visit Google AI Studio and sign in with your Google account. Click "Create API Key" and copy it. The free tier includes generous usage limits.

Is my data safe?

Absolutely. Spokable PDF runs entirely in your browser. Your PDF files are processed locally and never uploaded to any server. Only the extracted text is sent to Google's Gemini API for transformation. Your API key is stored locally in your browser's storage.

What file size limits exist?

There's no hard limit imposed by the app itself. However, very large PDFs (100+ pages) may take longer to process and consume more API quota. The app uses intelligent batching to handle large documents efficiently.

Which AI model should I use?

For most users, Gemini 2.5 Flash (default) offers the best balance of speed, quality, and cost. Use Gemini 3 Pro Preview for maximum quality on complex technical documents. Use Flash Lite for faster processing of simpler documents.

What is Turbo Mode?

Turbo Mode processes multiple batches of your PDF simultaneously using parallel API requests. This significantly speeds up conversion but may hit rate limits faster. Disable it if you encounter frequent rate limit errors.

How much does it cost?

The app itself is completely free and open source. You only pay for Google Gemini API usage based on your consumption. The free tier includes 15 requests per minute and 1,500 requests per day, which is sufficient for most users. See pricing details.

Can I customize the conversion prompts?

Yes! Go to Settings and scroll to "Prompt Customization". You can modify how code blocks, tables, math notation, and general text are transformed. This allows you to tailor the output to your specific needs.

What happens if processing fails?

The app includes robust error handling with automatic retries. If a batch fails due to rate limits, it will retry with exponential backoff. If you've configured a backup API key, it will automatically failover. You can also download partially processed PDFs if needed.

Does it work offline?

The app interface works offline once loaded, but PDF conversion requires an internet connection to communicate with the Gemini API. The actual PDF processing (extraction and generation) happens locally in your browser.

Which TTS apps work best with Spokable PDFs?

Spokable PDFs work with any TTS application. Popular choices include Moon+ Reader Pro (Android), Voice Dream Reader (iOS), Natural Reader, and built-in accessibility features in Adobe Acrobat and web browsers.

Can I process scanned PDFs?

Currently, the app works best with text-based PDFs. Scanned PDFs (images of text) require OCR (Optical Character Recognition) first. Consider using Adobe Acrobat or online OCR tools to convert scanned PDFs to text-based PDFs before processing.

How accurate is the AI transformation?

Google Gemini is highly accurate for technical content. However, AI can occasionally misinterpret complex notation or context. Always review critical documents. You can adjust prompts in Settings to improve accuracy for your specific use case.

Can I contribute to the project?

Yes! Spokable PDF is open source. Visit the GitHub repository to report bugs, suggest features, or submit pull requests.

What browsers are supported?

Spokable PDF works on all modern browsers that support ES6+ JavaScript, IndexedDB, and the File API. This includes Chrome, Firefox, Safari, and Edge (latest versions).
